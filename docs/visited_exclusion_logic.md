# 이미 가본 곳 제외 로직

작성일: 2026-05-13

## 원칙

이미 가본 곳 제외는 로그인이나 저장 없이 현재 세션에서만 동작한다. 새로고침하거나 세션이 초기화되면 제외 목록도 초기화된다.

## 설정 위치

설정 섹션은 페이지 맨 아래에 둔다. 상단 필터와 빠른 선택 패널의 `이미 가본 곳 제외 설정` 버튼을 누르면 하단 섹션이 펼쳐진다.

## 입력 방식

1. 가본 장소명 선택
2. 가본 프로그램명 선택
3. 장소나 프로그램 이름 직접 입력

## 체크 옵션

- 같은 장소 제외
- 같은 프로그램 제외
- 같은 공식 링크 제외

기본값은 모두 선택이다.

## 적용 방식

`이 설정으로 다시 추천 보기`를 누르면 현재 추천 결과에서 해당 후보를 제외한다. 제외된 후보에는 `이미 가본 곳` 이유가 붙는다.

## 제외 후보 복구

`제외된 후보 보기`에서 개별 후보의 `다시 포함` 버튼을 누르면 세션 제외 목록에서 해당 후보를 제거한다.

## 완료 메시지

`이미 가본 곳 N개를 제외하고 추천을 다시 계산했습니다.`
