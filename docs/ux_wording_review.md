# 부모용 화면 문구 검토

## 원칙

부모용 화면에서는 분석·정책·개발 버전 용어를 줄이고, 실제 의사결정에 가까운 말을 사용한다.

## 대체 문구

| 기존 표현 | 부모용 표현 |
|---|---|
| 플랜B | 대체 후보 / 막혔을 때 갈 만한 곳 |
| 조건 완화 퍼널 | 조건을 넓히면 늘어나는 후보 |
| 데이터 품질 패널 | 데이터 확인 |
| 선택가능도 | 갈 수 있는 후보 수 |
| 정책 인사이트 | 서울시 관점 / 지역별 기회 보기 |
| API 확인 | 공공데이터 확인 |
| 링크 확인 필요 | 공식 링크에서 최종 확인 |
| 후보 329개 | 조건에 맞는 후보 더보기 |

## 추천 카드 문구

- 오늘의 추천
- 왜 이곳인가?
- 주의할 점
- 막혔을 때 갈 만한 곳
- 지금은 추천하지 않은 이유
- 조건을 넓히면 더 보입니다
- 공식 링크에서 신청 가능 여부 확인

## 금지 문구

부모용 화면과 제출용 설명에서는 이전 버전명, 내부 참조명, 구현 관점의 문구를 노출하지 않는다.
