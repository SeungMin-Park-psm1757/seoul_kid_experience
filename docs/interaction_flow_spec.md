# 상호작용 흐름 규격

작성일: 2026-05-13

## 기본 사용자 흐름

1. 사용자가 아이 나이, 학년, 출발 자치구, 날짜, 이동시간, 비용, 실내/실외, 관심분야를 설정한다.
2. `추천 결과 보기`를 누른다.
3. 부모용 홈에서 `오늘의 추천 1곳`을 먼저 확인한다.
4. 추천 카드에서 `공식 링크에서 확인`, `상세 보기`, `대체 후보 보기`, `이미 가본 곳으로 표시` 중 다음 행동을 선택한다.
5. 상세 또는 대체 후보가 필요하면 `상세 확인` 탭에서 이어서 확인한다.

## 필터 하단 CTA

- `추천 결과 보기`: 현재 필터 기준으로 결과를 다시 계산했다는 안내를 표시하고 오늘의 추천을 먼저 보여준다.
- `조건 초기화`: 필터와 이미 가본 곳 제외 목록을 기본값으로 복원한다.
- `이미 가본 곳 제외 설정`: 하단의 제외 설정 섹션을 펼친다.

## 빠른 선택 패널

부모용 홈 상단의 `다음에 무엇을 볼까요?`는 버튼형 선택 흐름이다.

- `추천 1곳`
- `추천 후보 5곳`
- `대체 후보`
- `추천하지 않은 이유`
- `조건 넓히기`
- `전체 지도`
- `이미 가본 곳 제외`
- `전체 보기`

## 이미 가본 곳 제외 흐름

이미 가본 곳 제외 설정은 페이지 맨 아래에 둔다. 상단 버튼을 누르면 `show_visited_section=True`가 되어 하단 expander가 열린다.

설정 항목:

- 가본 장소명
- 가본 프로그램명
- 장소나 프로그램 이름 직접 입력
- 같은 장소 제외
- 같은 프로그램 제외
- 같은 공식 링크 제외

설정 후 `이 설정으로 다시 추천 보기`를 누르면 현재 세션에서 제외 목록을 반영해 추천을 다시 계산한다.

## 카드 CTA

오늘의 추천 카드:

- 공식 링크에서 확인
- 이 후보 상세 보기
- 대체 후보 보기
- 이미 가본 곳으로 표시

추천 후보 카드:

- 공식 링크에서 확인
- 상세 보기
- 대체 후보
- 제외하기

신청 어려운 후보 카드:

- 공식 링크에서 확인
- 왜 제외됐는지 보기
- 대체 후보
- 제외하기
