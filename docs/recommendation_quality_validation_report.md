# 추천 품질 보완 검증 리포트

검증일: 2026-05-13

## 문제

부모용 추천에서 다음 문제가 확인되었다.

- 같은 장소와 같은 프로그램 시리즈가 월/회차만 달라 여러 장 반복 노출됨
- `누구나`, `0~99세`, `교육/체험`으로 들어온 성인 교양·문학·역사 주제가 아이 추천에 포함됨
- 화면 필터에는 학년이 있지만, 추천 로직은 일부 프로그램의 `3~6학년` 같은 학년 제한을 충분히 반영하지 못함

## 적용한 보완

| 영역 | 보완 내용 |
|---|---|
| 아이 콘텐츠 난이도 | 7~9세 기준으로 `5.18`, `광주`, `증언`, `전쟁`, `참사`, `폭력` 등 무거운 주제가 있고 어린이/가족 맥락이 없으면 기본 추천에서 제외 |
| 일반 대상 검토 | `누구나` 또는 `제한없음`이더라도 어린이/초등/가족 맥락이 없고 북콘서트·강연·세미나 등 성인 교양 성격이면 기본 추천에서 제외 |
| 학년 제한 | `3~6학년`, `초등 1~3학년` 같은 문구를 파싱해 현재 선택 학년과 맞지 않으면 추천 제외 |
| 중복 시리즈 | 제목에서 월, 회차, 정원, 날짜, 요일 표기를 제거한 `series_key`를 만들고 같은 장소·같은 카테고리·같은 시리즈는 1개만 노출 |

## 실제 확인 사례

| 사례 | 수정 후 처리 |
|---|---|
| `문예북송 No.61 - 『5.18 푸른 눈의 증인』 ... 오월의 광주` | 초등 저학년에게 무거운 주제로 분류, 기본 추천 제외 |
| `(토,일) 5월 불암산나비정원 곤충아카데미-장수풍뎅이 표본교실-` 및 6월 동일 시리즈 | 같은 시리즈로 묶어 Top 5 중복 노출 방지 |
| `(3~6학년) Guro 월드컵` | 선택 학년이 초2이면 학년 조건 불일치로 기본 추천 제외 |

## 남은 한계

- 콘텐츠 난이도는 공개 제목·대상·설명 문구 기반의 보수적 신호다.
- 실제로 아이에게 적합한지는 보호자가 공식 링크에서 세부 내용을 확인해야 한다.
- 같은 시리즈 판정은 제목 정규화 기반이므로, 제목이 크게 다른 동일 프로그램은 일부 남을 수 있다.
