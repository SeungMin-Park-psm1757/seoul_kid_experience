# 예약 난이도 신호

## 목적

공개 API만으로 실제 회차별 잔여석을 확정할 수 없으므로, 부모에게 “예약 가능 확정” 대신 공개 상태 기반의 예약 난이도 신호를 제공한다.

## 값

- `easy`: 예약 부담 낮음
- `medium`: 확인 필요
- `hard`: 마감 위험 높음
- `unknown`: 공식 확인 필요

## 산정 기준

`easy`:

- 상시
- 현장
- 자유이용
- 예약 불필요
- 문화행사 또는 상설 공간형 후보

`medium`:

- 접수중 또는 예약가능이나 회차 확인 필요
- 접수 예정
- 정원 제한은 있으나 마감 문구 없음

`hard`:

- 마감임박
- 접수마감
- 예약마감
- 정원마감
- 잔여석 부족, 조기마감, 선착순, 대기, 소수정원 문구

`unknown`:

- 상태, 날짜, 정원 문구가 불명확한 경우

## 향후 확장

현재 `historical_pattern_available=False`다. 매일 API 스냅샷이 쌓이면 다음 지표를 추가할 수 있다.

- 동일 SVCID 상태 변화
- 접수중에서 마감까지 걸린 일수
- 반복 조기마감 후보
- 카테고리별 마감 속도

현재 화면에서는 “예약 난이도 신호는 현재 공개 상태와 일정 기준입니다. 반복 스냅샷이 쌓이면 마감 패턴 분석으로 확장할 수 있습니다.”라고 설명한다.
