# 대상연령·대상층 하드필터 규칙

## 목적

부모용 추천 화면에서 청년·성인·반려견·조직 모집 프로그램이 7세/초등 저학년 후보로 노출되지 않도록 대상층을 점수가 아니라 절대 필터로 처리한다.

## 허용 대상

- `child`: 어린이, 아동, 아이 대상
- `family`: 가족, 부모·보호자 동반 맥락
- `infant`: 유아 대상이며 선택 아이 나이가 맞는 경우
- `elementary`: 초등학생 대상
- `general_public`: 제한없음이지만 제목·분류·장소가 아동/가족/체험 맥락인 경우

## 기본 제외 대상

- `youth`: 청년, 대학생, 취업·창업·서포터즈 등
- `adult`: 성인, 직장인, 강사, 활동가, 자원봉사 등
- `senior`: 어르신, 시니어, 노인, 노약자
- `pet`: 반려견, 반려동물, 강아지, 펫 등
- `organization`: 위원회, 네트워크, 정책참여, 모집 등
- `unknown`: 부모용 추천에서는 제외하고 확인 필요 영역에서만 설명

## 키워드 예외

`반려견`, `동물` 키워드가 있더라도 `어린이 동물보호교육`처럼 제목과 대상 문구에 어린이·초등·가족 맥락이 있으면 제외하지 않는다.

## 제한없음 처리

`제한없음`, `0~99세`는 자동 적합 판정하지 않는다. 아동·가족·체험 맥락이 확인될 때만 추천 후보로 허용하고, 그렇지 않으면 `review_needed` 또는 `excluded`로 분류한다. 다만 명시적 어린이·가족·초등 후보가 있으면 넓은 일반 관람 후보보다 먼저 정렬한다.

## 산출 필드

- `target_audience_raw`
- `target_audience_norm`
- `target_audience_confidence`
- `child_suitability`
- `child_suitability_reason`
- `hard_filter_reason`
