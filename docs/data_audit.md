# 데이터 감사 기록

현재 모드: 샘플 데이터

data/raw 상태: 원본 파일 없음. 현재는 샘플 데이터만 사용.

## 현재 처리 상태
- `python -m src.build_all --sample`로 샘플 기반 처리 CSV와 5개 시각화를 생성했다.
- 실제 원본 CSV가 `data/raw/`에 들어오면 `src/normalize.py`의 컬럼 후보 또는 데이터셋별 mapping 함수를 맞춰야 한다.
- 원본에 실시간 잔여석이 없을 수 있으므로 예약 가능성은 서비스상태·운영기간·접수기간 기반 탐색 지표로만 사용한다.
