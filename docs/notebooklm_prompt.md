# NotebookLM용 PPT 초안 생성 프롬프트

다음 자료들을 바탕으로 공모전 시각화 부문 제출용 PPT 초안을 작성해줘.

주제명: 서울 데이터 기반 아이체험 큐레이터
부제: 주말 공공체험의 선택가능도와 플랜B 격차를 보여주는 서울 아동 생활권 시각화

중요:
이 자료는 아이체험 추천 앱 소개서가 아니라, 서울 데이터 허브 공공데이터를 활용한 시각화 부문 제출자료다. 따라서 서비스 기능보다 문제정의, 데이터 활용, 지표 설계, 핵심 시각화 결과, 정책 인사이트를 중심으로 구성해줘.

반드시 포함할 핵심 결과:
1. 빈 주말 캘린더
2. 플랜B 충분도 지도 또는 타일맵
3. 연령별 블라인드 스팟 히트맵
4. 선택가능도 × 플랜B 충분도 2×2 매트릭스
5. 이용장벽 분해 차트

각 장마다 다음 항목을 포함해줘.
- 슬라이드 제목
- 핵심 메시지
- 본문 문장
- 들어갈 시각화 또는 표
- 발표자가 말할 포인트
- 심사기준 중 어떤 항목에 대응하는지

주의:
- 로그인, 회원관리, 예약 자동화처럼 보이지 않게 해줘.
- 노쇼는 발생률 분석이 아니라 이용규칙 가시성 문제로만 다뤄줘.
- 웹 대시보드는 서비스 개발이 아니라 인터랙티브 시각화 결과물로 설명해줘.
