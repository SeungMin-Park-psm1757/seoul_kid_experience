# 프로젝트 브리프

## 프로젝트명
서울 데이터 기반 아이체험 큐레이터

## 부제
주말 공공체험의 선택가능도와 플랜B 격차를 보여주는 서울 아동 생활권 시각화

## 핵심 문제정의
서울에는 아이와 갈 수 있는 공공체험이 존재하지만, 부모는 연령·날짜·비용·거리·예약상태·이용규칙을 따로 확인해야 한다.  
문제는 단순한 체험 부족이 아니라, 자치구·연령·시기별 **선택가능도와 플랜B 격차**다.

## 핵심 질문
1. 어느 자치구는 주말 아이체험 선택지가 부족한가?
2. 첫 선택이 막혔을 때 플랜B가 남아 있는가?
3. 특정 연령대가 구조적으로 배제되는 지역은 어디인가?

## 최종 산출물의 정체성
- 앱/서비스 개발물이 아니라 **공공데이터 기반 인터랙티브 시각화 결과물**
- 부모의 선택 순간과 서울시 정책 판단을 동시에 지원하는 시각화
- 기능보다 데이터 해석과 정책 인사이트가 중심

## 제외 범위
- 로그인
- 회원관리
- 예약 자동화
- 실시간 취소표 추적
- 노쇼 발생률 분석
- 결제/알림/푸시

## 포함 범위
- 데이터 통합
- 지표 산출
- 자치구/연령대/주차별 시각화
- 플랜B 대체가능성 분석
- 이용장벽 가시화
- 인터랙티브 대시보드
