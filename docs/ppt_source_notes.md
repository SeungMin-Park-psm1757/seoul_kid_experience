- 특정 연령대는 동일한 서울 안에서도 공공체험 선택권이 제한될 수 있다.

## 발표 시 주의 문구
- 지표는 공공데이터 기반 탐색적 지표이며 정책 판단에는 추가 검증이 필요하다.
- 실시간 예약 잔여석이나 노쇼 발생률을 분석한 결과로 표현하지 않는다.
- 노쇼/취소 관련 문구는 이용규칙 가시성 또는 이용장벽 요소로만 설명한다.
