# 핵심 시각화 설계서

## 1. 빈 주말 캘린더

### 질문
언제, 어디서 아이체험 선택권이 비는가?

### 형태
- 행: 자치구
- 열: 1~52주
- 색상: 선택가능도 지수
- 강조: 하위 20% 또는 특정 임계값 이하 주말

### PPT 역할
작품의 시그니처 뷰. “아이체험판 DARK WEEKS” 역할.

## 2. 플랜B 충분도 지도/타일맵

### 질문
첫 선택이 막혔을 때, 어느 지역에는 대체안이 남고 어느 지역에는 사라지는가?

### 형태
- 자치구별 색상: 플랜B 충분도
- 보조 랭킹: 상위/하위 자치구
- 실제 지도가 없으면 25개 자치구 tilemap으로 대체

### PPT 역할
승민님 아이디어의 차별점인 플랜B를 정책 지표로 전환.

## 3. 연령별 블라인드 스팟

### 질문
모든 연령대가 같은 공공체험 선택권을 갖는가?

### 형태
- 4~6세, 7~9세, 10~12세별 스몰멀티플 지도 또는 히트맵
- 각 자치구별 선택가능도 차이 표시

### PPT 역할
“아이체험”을 더 세밀하게 만드는 분석 장면.

## 4. 선택가능도 × 플랜B 충분도 2×2 매트릭스

### 질문
어느 자치구가 정책적으로 가장 우선 지원되어야 하는가?

### 형태
- X축: 선택가능도
- Y축: 플랜B 충분도
- 점: 자치구
- 4유형: 풍부/취약 조합

### PPT 역할
정책 우선순위 도출.

## 5. 이용장벽 분해 차트

### 질문
있어도 고르기 어려운 체험은 무엇 때문에 어려운가?

### 형태
- 자치구별 stacked bar 또는 상위/하위 비교
- 비용, 예약, 연령 제한, 규칙 가시성 요소 분해

### PPT 역할
노쇼/취소/규칙 문제의식을 안전하게 반영.
