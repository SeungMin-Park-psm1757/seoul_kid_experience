# 부모 사용자 최종 여정

작성일: 2026-05-13

## 목표

부모가 “해당 날짜에 아이와 어디 갈지”를 빠르게 판단하도록, 조건 입력부터 공식 링크 확인까지의 클릭 흐름을 명확히 한다.

## 기본 흐름

1. 조건을 고른다.
2. `추천 결과 보기`를 누른다.
3. `오늘의 추천 1곳`을 먼저 확인한다.
4. 마음에 들면 `공식 링크에서 확인`을 누른다.
5. 더 알고 싶으면 `이 후보 상세 보기`를 누른다.
6. 이미 가본 곳이면 `이미 가본 곳으로 표시`를 누른다.
7. 마감이거나 마음에 들지 않으면 `대체 후보 보기`를 누른다.
8. 그래도 후보가 부족하면 `조건 넓히기`를 본다.
9. 더 많이 보고 싶으면 `전체 지도` 또는 `추천 후보 5곳`을 본다.

## 화면별 역할

- 부모용 홈: 오늘의 추천, 추천 후보 5곳, 대체 후보, 조건 넓히기, 추천하지 않은 이유
- 추천 탐색: 지도와 후보 리스트
- 후보 비교: 오늘의 추천과 대체 후보 1~2개 비교
- 상세 확인: 추천 이유, 주의할 점, 이용규칙, 공식 링크, 대체 후보
- 서울시 관점: 지역별 기회와 이용장벽 분석
- 이미 가본 곳 제외 설정: 현재 세션에서만 제외할 장소/프로그램 선택

## 핵심 CTA

- `추천 결과 보기`: 필터 조건 반영
- `공식 링크에서 확인`: 공식 예약/출처 페이지 확인
- `이 후보 상세 보기`: 상세 확인 탭에서 후보 확인
- `대체 후보 보기`: 첫 후보가 어려울 때 볼 후보 확인
- `이미 가본 곳으로 표시`: 현재 세션 추천에서 제외
- `이 설정으로 다시 추천 보기`: 제외 조건 반영 후 재계산

## 주의 원칙

이 앱은 예약 서비스가 아니다. 실제 신청 가능 여부와 회차별 잔여석은 공식 링크에서 최종 확인해야 한다.
