## 남은 한계

- 공공 API의 접수상태는 실제 회차별 신청 성공을 보장하지 않는다.
- 상세 페이지 파싱은 공개 HTML 구조에 의존하므로 실패할 수 있다.
- 이동시간은 실제 교통 API가 아니라 자치구/좌표 기반 추정이다.
- 예약 난이도는 현재 공개 상태와 일정 기준 신호이며, 반복 스냅샷이 쌓인 뒤에야 마감 패턴 분석으로 확장할 수 있다.
