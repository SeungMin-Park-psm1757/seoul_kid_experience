# 주말·요일 하드필터 규칙

## 목적

사용자가 토요일 또는 일요일을 선택했을 때 평일 전용 프로그램이 오늘의 추천, Top 5, 대체 후보에 들어가지 않도록 한다.

## 주말 허용 키워드

- 토
- 토요일
- 일
- 일요일
- 토,일
- 주말
- 공휴일
- 상시
- 연중
- 매일

## 평일 전용 키워드

- 평일
- 평일주간
- 평일야간
- 월~금
- 월-금
- 월요일
- 화요일
- 수요일
- 목요일
- 금요일
- 주중

## 처리 방식

선택 날짜가 주말이면 `is_weekday_only=True`인 항목은 부모용 추천과 대체 후보에서 제외한다. `is_weekend_available=True`이거나 운영 기간이 선택 날짜를 포함하면 기본 후보로 남길 수 있다. 요일 정보가 불명확한 항목은 기본 추천이 아니라 확인 필요 영역으로 보낸다.

## 사용자 문구

- “선택한 날짜가 주말이라 평일 전용 프로그램은 제외했습니다.”
- “평일 프로그램까지 보려면 날짜 조건을 바꿔 주세요.”
- “요일 정보가 불명확해 공식 링크에서 최종 확인이 필요합니다.”
