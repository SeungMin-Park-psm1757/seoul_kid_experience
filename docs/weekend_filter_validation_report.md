## 적용한 수정

- `src/schedule_parser.py`에서 프로그램명, 시간, 대상, 규칙, 원본 `raw_*` 필드의 운영 요일 문구를 파싱한다.
- `src/recommend.py`에서 오늘의 추천, 추천 후보 5곳, 대체 후보, 지도 기본 결과에 `is_selected_date_eligible=True` 조건을 하드필터로 적용한다.
- `date_start~date_end`가 선택 날짜를 포함하더라도, 명시 운영 요일이 선택 날짜와 다르면 추천 점수를 계산하지 않는다.
- 요일이 불명확한 장기 운영 항목은 기본 추천이 아니라 “공식 확인 필요” 또는 “지금은 추천하지 않은 이유” 흐름으로 분리한다.

## 카드 배지 검증

- `date_start~date_end` 기간에 포함된다는 이유만으로 `날짜 일치` 배지를 붙이지 않도록 수정했다.
- 명시 요일이 선택 날짜와 맞으면 `운영 요일 일치`를 표시한다.
- 같은 주말의 다른 날짜에 운영되는 항목은 `같은 주말`로 표시한다.
- 요일 정보가 불명확하면 `공식 확인 필요`로 분리한다.
