# Codex 프롬프트 02 — 실제 서울 데이터 연결

`data/raw/`에 실제 서울 데이터 허브/열린데이터광장 CSV 또는 JSON 파일을 넣었다고 가정한다.

작업:
1. 각 파일의 컬럼명을 읽고 `docs/data_audit.md`를 작성하라.
2. `src/normalize.py`에 실제 컬럼명 매핑을 추가하라.
3. 모든 데이터를 `unified_programs.csv` 스키마에 맞게 통합하라.
4. 실제 데이터 기반으로 지표를 다시 계산하라.
5. 샘플 기반 문구가 남아 있으면 제거하라.
6. `docs/metrics_result_summary.md`에 상위/하위 자치구, 연령대별 차이, 빈 주말 패턴을 요약하라.

주의:
- 좌표가 없는 데이터는 자치구 단위 집계에만 사용하라.
- 실시간 잔여석이 없으면 실시간 가능성을 주장하지 말라.
- 노쇼 발생률 분석을 하지 말라.
